https://resources.rosemaryorchard.com/images/blog/workflow-taskpaper-to-omnifocus/workflow-taskpaper-to-of.png "Taskpaper To OF"){.image-right} This is a very simple Workflow which allows you to add TaskPaper formatted tasks or projects to your preferred place in OmniFocus

Steps:

  • https://resources.rosemaryorchard.com/images/workflow-icons/Calculator.png "Count") Count: This checks for input.
  • https://resources.rosemaryorchard.com/images/workflow-icons/Scripting.png "If") If: If there is no input...
    • https://resources.rosemaryorchard.com/images/workflow-icons/Text.png "Text") Text: Show you your clipboard and allow you to edit it.
  • https://resources.rosemaryorchard.com/images/workflow-icons/Scripting.png "If") Else
    • https://resources.rosemaryorchard.com/images/workflow-icons/Variable.png "Get Variable") Get Variable: Get the input you provided to the Workflow
  • Add TaskPaper to OmniFocus: This asks you to choose where you want to add the TaskPaper: Inbox, Projects, Folder, Project, or Task.

You can get the workflow here: Taskpaper To OF
