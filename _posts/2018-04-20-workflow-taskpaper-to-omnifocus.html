https://resources.rosemaryorchard.com/images/blog/workflow-taskpaper-to-omnifocus/workflow-taskpaper-to-of.png?cropResize=100 "Taskpaper To OF"){.image-right} This is a very simple Workflow which allows you to add TaskPaper formatted tasks or projects to your preferred place in OmniFocus

Steps:

  • https://resources.rosemaryorchard.com/images/workflow-icons/Calculator.png?cropResize=25 "Count") Count: This checks for input.
  • https://resources.rosemaryorchard.com/images/workflow-icons/Scripting.png?cropResize=25 "If") If: If there is no input...
    • https://resources.rosemaryorchard.com/images/workflow-icons/Text.png?cropResize=25 "Text") Text: Show you your clipboard and allow you to edit it.
  • https://resources.rosemaryorchard.com/images/workflow-icons/Scripting.png?cropResize=25 "If") Else
    • https://resources.rosemaryorchard.com/images/workflow-icons/Variable.png?cropResize=25 "Get Variable") Get Variable: Get the input you provided to the Workflow
  • Add TaskPaper to OmniFocus: This asks you to choose where you want to add the TaskPaper: Inbox, Projects, Folder, Project, or Task.

You can get the workflow here: Taskpaper To OF
