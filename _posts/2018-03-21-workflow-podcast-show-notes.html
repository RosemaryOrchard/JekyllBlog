00 "Mac Power Users"){.image-right} This is a very simple Workflow I use to get the show notes for the Mac Power Users podcast, I use it when I’m listening on another device and I don’t want to grab it or interrupt what I’m doing. It’s quite simple, just 3 steps in fact.

Steps:

  • https://resources.rosemaryorchard.com/images/workflow-icons/Scripting.png "Scripting") Ask for Input: This asks you for the episode number.
  • https://resources.rosemaryorchard.com/images/workflow-icons/URL.png "URL") URL This creates the URL for the show notes.
  • https://resources.rosemaryorchard.com/images/workflow-icons/Web.png "Web") Open URLs And this opens the above URL.

You can get the workflow here: Mac Power Users Show Notes. It’s designed to run in the today widget.

https://resources.rosemaryorchard.com/images/blog/workflow-podcast-show-notes/workflow-podcast-show-notes.png "Podcast Show Notes"){.image-right} Now, you might listen to more than one podcast (dramatic music) - and you probably don’t want one of these for every podcast you listen to, that’s just not manageable. But with some help from the dictionary action you can make it much easier.

Steps:

  • https://resources.rosemaryorchard.com/images/workflow-icons/Scripting.png "Scripting") Dictionary: This is where you store the list of the friendly name for the podcast and the URL which comes before the show number, so in the case of the Mac Power Users podcast, that is http://www.relay.fm/mpu/ - attaching an episode number (such as 318) gives you a link directly to the show notes.
  • https://resources.rosemaryorchard.com/images/workflow-icons/Scripting.png "Scripting") Ask for Input: This asks you for the episode number.
  • https://resources.rosemaryorchard.com/images/workflow-icons/Variable.png "Variable") Get Variable: This grabs the dictionary from above.
  • https://resources.rosemaryorchard.com/images/workflow-icons/Scripting.png "Scripting") Choose from List: Here’s where you choose the podcast you’re listening to.
  • https://resources.rosemaryorchard.com/images/workflow-icons/URL.png "URL") URL This creates the URL for the show notes.
  • https://resources.rosemaryorchard.com/images/workflow-icons/Web.png "Web") Open URLs And this opens the above URL.

You can get the workflow here: Podcast Show Notes
